9. 페이지 로딩 작업 만들기

9_1. 페이지 로딩 작업 만들기

페이지를 클릭할 때마다 메인 콘텐츠에 트랜지션 효과를 추가하겠습니다. 데이터는 로컬로 되어 있기 때문에 다운로딩 시간은 별로 걸리지 않을 것입니다. 하지만 사이트의 UX 개선을 위해 페이지를 클릭할 때마다 트랜지션 효과를 추가하여 페이지 전환 느낌을 추가하겠습니다.

우선 Webd.jsx부터 시작해보겠습니다. 

import React, { useEffect, useState } from 'react'
import Main from '../components/section/Main'

import VideoCards from '../components/videos/VideoCards'
import { webdText } from '../data/webd'

const Webd = () => {
    const [loading, setLoading] = useState(true); 

    useEffect(() => {
        setTimeout(() => {
            setLoading(false);
        }, 300);
    }, []);

    const webdPageClass = loading ? 'isLoading' : '';

    return (
        <Main 
            title = "웹디자인 기능사"
            description="웹디자인 기능사 튜토리얼 강의입니다.">

            <section id='webdPage' className={webdPageClass}>
                <h2>😄 웹디자인기능사 한번에 따자!</h2>
                <div className="video__inner">
                    {!loading && <VideoCards videos={webdText} /> }
                </div>
            </section>
        </Main>
    )
}

export default Webd
  • useState 및 useEffect 사용: useState와 useEffect 훅을 사용하여 상태(loading)를 관리하고, 컴포넌트가 마운트된 후 일정 시간이 지난 후에 로딩 상태를 false로 변경합니다. 이를 통해 화면에서 로딩 상태를 관리할 수 있습니다.
  • Main 컴포넌트: Main 컴포넌트는 웹 페이지의 주요 섹션을 렌더링합니다. 이 컴포넌트는 title과 description props를 받아 페이지의 제목과 설명을 설정합니다.
  • 웹디자인 페이지 섹션: 웹디자인 페이지에 대한 섹션을 정의합니다. webdPage라는 ID와 isLoading 클래스를 조건에 따라 추가합니다. 로딩 상태(loading)가 true일 때는 페이지에 "isLoading" 클래스를 추가하여 로딩 스타일을 적용하며, 로딩이 완료되면 이 클래스를 제거합니다.
  • VideoCards 컴포넌트: VideoCards 컴포넌트는 비디오 카드 목록을 표시합니다. 로딩이 완료된 후에 VideoCards 컴포넌트를 렌더링합니다. webdText 데이터를 videos props로 전달하여 비디오 목록을 표시합니다.

VideoSlider.jsx를 변경해주겠습니다. 

import React, { useEffect, useState } from 'react'
import { Link } from 'react-router-dom'

import { Swiper, SwiperSlide } from 'swiper/react';
import 'swiper/css';
import 'swiper/css/navigation';

import { Navigation } from 'swiper/modules';

const VideoSlider = ({ videos, title, id }) => {
    const [loading, setLoading] = useState(true); 

    useEffect(() => {
        setTimeout(() => {
            setLoading(false);
        }, 300);
    }, []);

    const youtubeSecionClass = loading ? 'isLoading' : '';

    return (
        <section id={id} className={youtubeSecionClass}>
            <h2>{title}</h2>

            <div className='video__slider'>
                <Swiper
                    slidesPerView={1}
                    spaceBetween={20}
                    navigation={true}
                    modules={[Navigation]}
                    className={`mySwiper-${id}`}
                    breakpoints={{
                        640: {
                            slidesPerView: 2,
                            spaceBetween: 20,
                        },
                        768: {
                            slidesPerView: 3,
                            spaceBetween: 20,
                        },
                        1024: {
                            slidesPerView: 4,
                            spaceBetween: 20,
                        },
                        1600: {
                            slidesPerView: 4,
                            spaceBetween: 20,
                        },
                    }}
                >
                    {videos.map((video, key) => (
                        <SwiperSlide key={key}>
                            <div className="video">
                                <div className="video__thumb play__icon">
                                    <Link to={`/video/${video.videoId}`}>
                                        <img src={video.img} alt={video.title} />
                                    </Link>
                                </div>
                            </div>
                        </SwiperSlide>
                    ))}
                </Swiper>
            </div>
        </section>
    )
}

export default VideoSlider

9_2. scrollTo 만들기

페이지가 전환이 될 때 새로운 페이지 위쪽에서 변경이 되야 하는데, 가끔씩 보면 중간에서 업로드 되는 경우가 있습니다. 이 때 스크립트를 통해 페이지가 전환되면 상단으로 올라올 수 있도록 설정하겠습니다.

utils 폴더를 만들고 scrollTo 파일을 만들겠습니다. 

import { useEffect } from 'react';
import { useLocation } from 'react-router-dom';

const ScrollTo = () => {
    const { pathname } = useLocation();

    useEffect(() => {
        window.scrollTo(0, 0);
    }, [pathname]);

    return null;
};

export default ScrollTo;
  • useLocation 훅을 사용하여 현재 경로 정보를 가져옵니다. pathname 변수에 현재 경로가 저장됩니다.
  • useEffect 훅은 컴포넌트가 렌더링될 때 실행되며, pathname이 변경될 때마다 다시 실행됩니다.
  • window.scrollTo(0, 0)은 브라우저의 스크롤 위치를 가로축(x)과 세로축(y) 모두 0으로 이동시킵니다. 즉, 페이지가 맨 위로 스크롤됩니다.
  • return null은 실제로 화면에 아무것도 렌더링하지 않음을 나타냅니다. 이 컴포넌트는 스크롤 관련 동작을 처리하기 위한 목적으로 사용되며 화면에는 아무 내용도 표시하지 않습니다.

소개

안녕하세요! 웹스토리보이입니다. 이 강의는 자바스크립트 초급부터 고급까지 아우르는 자바스크립트 핵심 개념을 다루는 기본서 강의입니다. 일반적인 자바스크립트 강의와는 목표와 방식이 다릅니다. 여기에서는 자바스크립트를 어떻게 단순하게 이해하고, 현실적으로 활용할 수 있는지를 고려하여 만들어진 내용입니다.

자바스크립트를 배우면서 때로는 이해하기 어려운 부분이나 무엇을 해야 하는지 혼란스러운 부분이 있을 수 있습니다. 그런 어려움을 극복하고 불필요한 시간 낭비를 줄이기 위해 이 강의는 차별화된 접근 방식을 채택하였습니다. 또한, 개인적으로도 어려움을 겪었던 부분들을 고려하여 설명하였습니다.

이 강의의 주요 목표는 자바스크립트를 학습하면서, 이론 뿐만 아니라 실제로 어떻게 활용할 수 있는지를 강조하는 것입니다. 이를 위해 강의는 크게 네 가지 주요 영역으로 구성되었습니다. 데이터 저장, 데이터 불러오기, 데이터 실행, 그리고 데이터 제어입니다. 이렇게 체계적으로 기초를 다지고 나서 실제 상황에 유용하게 활용할 수 있는 실용적인 예제를 다룰 것입니다. 이를 통해 프로그래밍 실력을 크게 향상시킬 수 있는 기회를 가지게 될 것입니다. 이 강의를 통해 자바스크립트의 세계에 더욱 가깝게 다가갈 수 있길 바랍니다.

3. 데이터 실행하기

01. 함수 : 선언적 함수

선언적 함수(Declaration function)는 function 키워드를 사용하여 정의된 함수입니다. 

{
}

func인 함수를 정의하고, 함수를 호출하여 함수 내부에서 변수 x를 선언하고 값을 출력하는 간단한 예제입니다.

  • function func() { ... } : 이 부분은 "func"라는 이름의 함수를 정의하는 부분입니다. 함수 정의는 function 키워드로 시작하며, 함수 이름 "func"가 나온 후에는 소괄호 ()로 감싸인 매개변수 목록이 올 수 있습니다. 이 예시에서는 매개변수가 없는 함수를 정의하고 있습니다. 중괄호 {} 내부에는 함수의 실행 코드가 위치합니다.
  • let x = 100; : 함수 내부에서 x라는 변수를 선언하고 값 100을 할당하고 있습니다. 이 변수는 함수 내부에서만 유효한 지역 변수입니다. 함수 내부에서 선언된 변수는 외부에서 접근할 수 없습니다.
  • console.log(x);: 함수 내부에서 선언한 변수 x의 값을 출력하고 있습니다. console.log() 함수는 브라우저 콘솔에 메시지를 출력하는 함수입니다. 따라서 이 코드는 함수가 호출되었을 때, 변수 x의 값인 100이 출력됩니다.
  • func();: 정의한 함수 func를 호출하고 있습니다. 함수 호출은 함수 이름 뒤에 소괄호 ()를 붙여서 이루어집니다. 함수 내부 코드가 실행되고, console.log(x);에 의해 변수 x의 값인 100이 콘솔에 출력됩니다.

3. 마무리

자바스크립트에서 데이터를 다루는 방법에 대해서 알아봤습니니다. 데이터를 저장하고 조작하는 것은 프로그래밍의 핵심 중 하나이며, 다양한 데이터 구조와 기법을 활용하여 프로그램을 더 효율적으로 작성할 수 있습니다. 자바스크립트에서 데이터를 다루는 방법은 크게 변수, 배열, 객체로 나눌 수 있습니다. 이러한 데이터 저장 방식을 통해 우리는 정보를 구조화하고 유용한 형태로 사용할 수 있습니다. 이런 데이터 조작 능력을 효과적으로 익히면, 자바스크립트 프로그래밍이 더욱 흥미로워질 것입니다. 그럼 오늘도 수고하셨습니다. 😊

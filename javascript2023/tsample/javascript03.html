안녕하세요! 웹스토리보이입니다. 이 강의는 자바스크립트 초급부터 고급까지 아우르는 자바스크립트 핵심 개념을 다루는 기본서 강의입니다. 일반적인 자바스크립트 강의와는 목표와 방식이 다릅니다. 여기에서는 자바스크립트를 어떻게 단순하게 이해하고, 현실적으로 활용할 수 있는지를 고려하여 만들어진 내용입니다.

자바스크립트를 배우면서 때로는 이해하기 어려운 부분이나 무엇을 해야 하는지 혼란스러운 부분이 있을 수 있습니다. 그런 어려움을 극복하고 불필요한 시간 낭비를 줄이기 위해 이 강의는 차별화된 접근 방식을 채택하였습니다. 또한, 개인적으로도 어려움을 겪었던 부분들을 고려하여 설명하였습니다.

이 강의의 주요 목표는 자바스크립트를 학습하면서, 이론 뿐만 아니라 실제로 어떻게 활용할 수 있는지를 강조하는 것입니다. 이를 위해 강의는 크게 네 가지 주요 영역으로 구성되었습니다. 데이터 저장, 데이터 불러오기, 데이터 실행, 그리고 데이터 제어입니다. 이렇게 체계적으로 기초를 다지고 나서 실제 상황에 유용하게 활용할 수 있는 실용적인 예제를 다룰 것입니다. 이를 통해 프로그래밍 실력을 크게 향상시킬 수 있는 기회를 가지게 될 것입니다. 이 강의를 통해 자바스크립트의 세계에 더욱 가깝게 다가갈 수 있길 바랍니다.

01. 함수 : 선언적 함수

선언적 함수(Declaration function)는 function 키워드를 사용하여 정의된 함수입니다. 

{
        function func(){
                console.log("1. 함수가 실행되었습니다.");
        }
        func();
}

func인 함수를 정의하고, 함수를 호출하여 함수 내부에서 변수 x를 선언하고 값을 출력하는 간단한 예제입니다.

  • function func() { ... } : 이 부분은 "func"라는 이름의 함수를 정의하는 부분입니다. 함수 정의는 function 키워드로 시작하며, 함수 이름 "func"가 나온 후에는 소괄호 ()로 감싸인 매개변수 목록이 올 수 있습니다. 이 예시에서는 매개변수가 없는 함수를 정의하고 있습니다. 중괄호 {} 내부에는 함수의 실행 코드가 위치합니다.
  • let x = 100; : 함수 내부에서 x라는 변수를 선언하고 값 100을 할당하고 있습니다. 이 변수는 함수 내부에서만 유효한 지역 변수입니다. 함수 내부에서 선언된 변수는 외부에서 접근할 수 없습니다.
  • console.log(x);: 함수 내부에서 선언한 변수 x의 값을 출력하고 있습니다. console.log() 함수는 브라우저 콘솔에 메시지를 출력하는 함수입니다. 따라서 이 코드는 함수가 호출되었을 때, 변수 x의 값인 100이 출력됩니다.
  • func();: 정의한 함수 func를 호출하고 있습니다. 함수 호출은 함수 이름 뒤에 소괄호 ()를 붙여서 이루어집니다. 함수 내부 코드가 실행되고, console.log(x);에 의해 변수 x의 값인 100이 콘솔에 출력됩니다.

02. 함수 : 익명 함수

익명 함수(Anonymous function)는 이름이 없는 함수입니다. 함수를 변수에 할당하여 사용하는 것은 함수를 다른 함수의 인자로 전달하거나, 함수를 반환하는 고급적인 활용도 가능하게 해줍니다. 

{
        const func = function(){
                console.log("2. 함수가 실행되었습니다.");
        }
        func();
}
  • const func = function() { ... } : 이 부분은 함수를 변수에 할당하는 익명 함수 표현식을 생성하고, 그 결과를 func라는 변수에 할당하고 있습니다. 변수에 할당된 함수는 해당 변수를 통해 호출되거나 전달될 수 있습니다.
  • function() { ... } : 이 부분은 익명 함수의 정의입니다. 이름이 없는 함수를 익명 함수라고 부릅니다. 변수 func에는 이 익명 함수의 참조가 저장됩니다. 이 부분에서도 function 키워드로 함수를 정의하고, 소괄호 ()로 매개변수를 받을 수 있습니다. 이 예시에서는 매개변수가 없는 함수를 정의하고 있습니다.
  • { ... } : 중괄호 내부에는 함수의 본문이 들어갑니다. 실행될 코드가 위치하며, 이 예시에서는 변수 x를 선언하고 값 100을 할당하고 있습니다.
  • console.log(x); : 함수 내부에서 선언한 변수 x의 값을 출력하고 있습니다. console.log() 함수는 브라우저 콘솔에 메시지를 출력하는 함수입니다.
  • func(); : 변수 func에 할당된 익명 함수를 호출하고 있습니다. 변수에 할당된 함수를 호출하려면 변수 이름 뒤에 소괄호 ()를 붙여야 합니다. 함수 내부 코드가 실행되며, console.log(x);에 의해 변수 x의 값인 100이 콘솔에 출력됩니다.

03. 함수 : 매개변수 함수

매개변수가 있는 함수는 매개변수를 통해 함수 내부로 값을 전달하여 동적인 처리를 할 수 있습니다. 

{
        // 선언적 함수
        function func(str){
                console.log(str);
        }
        func("3. 함수가 실행되었습니다.");

        // 익명 함수
        const func1 = function(str){
                console.log(str);
        }
        func("3. 함수가 실행되었습니다.");
}
  • function func(x) { ... }: 이 부분은 "func"라는 이름의 함수를 정의하는 부분입니다. 함수 정의는 function 키워드로 시작하며, 함수 이름 "func"가 나온 후에는 소괄호 ()로 매개변수 목록이 옵니다. 이 예시에서는 x라는 매개변수를 갖는 함수를 정의하고 있습니다. 매개변수는 함수 내에서 사용할 값을 받아올 수 있는 변수입니다.
  • { ... }: 중괄호 내부에는 함수의 본문이 들어갑니다. 실행될 코드가 위치하며, 이 예시에서는 매개변수 x를 출력하는 console.log(x);가 포함되어 있습니다.
  • func(100);: 함수 func를 호출하고 있습니다. 이때, 함수 호출 시에 100이라는 값이 인자로 전달됩니다. 함수 내에서 정의된 매개변수 x는 호출 시 전달된 값인 100을 받아옵니다.

04. 함수 : 리턴값 함수

리턴값 함수는 함수가 실행된 후에 반환하는 값을 말합니다. 

{
        // 선언적 함수
        function func(){
                return "4. 함수가 실행되었습니다.";
        }
        console.log(func());

        // 익명 함수
        const func1 = function(x){
                return "4. 함수가 실행되었습니다.";
        }
        console.log(func1());
}
  • function func(x) { ... }: 이 부분은 "func"라는 이름의 함수를 정의하는 부분입니다. 함수 정의는 function 키워드로 시작하며, 함수 이름 "func"가 나온 후에는 소괄호 ()로 매개변수 목록이 옵니다. 이 예시에서는 x라는 매개변수를 갖는 함수를 정의하고 있습니다.
  • { ... }: 중괄호 내부에는 함수의 본문이 들어갑니다. 실행될 코드가 위치하며, 이 예시에서는 매개변수 x를 그대로 반환하는 return x; 문이 포함되어 있습니다. 즉, 함수가 호출되면 매개변수 x의 값을 그대로 리턴합니다.
  • console.log(func(100));: 함수 func를 호출하고 있습니다. 이때, 함수 호출 시에 100이라는 값이 인자로 전달됩니다. 함수 내부로 전달된 100은 매개변수 x에 대입되며, 그대로 리턴됩니다. 이 리턴된 값, 즉 100,이 console.log() 함수에 의해 콘솔에 출력됩니다.

05. 함수 : 매개변수 + 리턴값 함수

                return str;
        }

        // 익명 함수

  • func가 매개변수로 100을 받아 호출되었습니다. 이 때, 함수 내부에서는 받은 값을 그대로 반환합니다. 반환된 값은 num이라는 변수에 할당됩니다.

06. 화살표 함수 : 선언적 함수

선언적 함수를 화살표 함수로 변경하는 예제입니다. 

{
        // 선언적 함수
        function func(){
                console.log("6. 함수가 실행되었습니다.");
        };
        func();

        // 선언적 함수 => 화살표 함수
        func = () => {
                console.log("6. 함수가 실행되었습니다.");
        }
        func();
}
  • func라는 이름의 화살표 함수를 정의합니다. 이 함수는 별도의 매개변수 없이 정의되었습니다. 함수 내부에서는 변수 x에 100이라는 값을 할당하고, 그 값을 콘솔에 출력하는 작업을 수행합니다.
  • 정의된 func 함수가 호출되었습니다. 이 때, 함수 내부의 코드가 실행됩니다.
  • 함수 내부에서 x 변수에 할당된 값인 100이 콘솔에 출력됩니다.

07. 화살표 함수 : 익명 함수

익명 함수를 화살표 함수로 변경하는 예제입니다. 

{
        // 익명 함수
        const func = func(){

10. 화살표 함수 : 매개변수 + 리턴값 함수

화살표 함수의 여러가지 형태와 약식 문법 예제입니다. 

{
        // 익명 함수 + 매개변수 + 리턴값 함수
        const func = (str) => {
                return str;
        }
        console.log(func("10. 함수가 실행되었습니다."));

        // 매개변수 한개일때 괄호 생략 가능
        const func2 = str => {
                return str;
        }
        console.log(func2("10. 함수가 실행되었습니다."));

        // 리턴 생략
        const func3 = str => str;
        
        console.log(func3("10. 함수가 실행되었습니다."));

        // 선언적 함수 (가독성X)
        func4 = str => str;
        
        console.log(func4("10. 함수가 실행되었습니다."));
}
  • 익명 함수 + 매개변수 + 리턴값 함수 : func에 익명 화살표 함수를 할당하고, 매개변수 x를 받아서 그 값을 반환하는 함수를 정의합니다. 이후 함수를 호출하여 매개변수로 100을 전달하고 반환값을 출력합니다.
  • 매개변수 한개일때 괄호 생략 가능 : 매개변수가 하나일 경우에는 괄호를 생략할 수 있는 화살표 함수를 정의합니다. 이후 함수를 호출하여 매개변수로 200을 전달하고 반환값을 출력합니다.
  • 리턴 생략 : 함수 내부에서 바로 값을 반환하는 경우에 리턴 키워드를 생략할 수 있는 화살표 함수를 정의합니다. 이후 함수를 호출하여 매개변수로 300을 전달하고 반환값을 출력합니다.
  • 선언적 함수 (가독성X) : 화살표 함수를 선언적으로 변수에 할당하는 형태로 정의합니다. 이 방식은 코드 가독성을 떨어뜨릴 수 있습니다. 이후 함수를 호출하여 매개변수로 400을 전달하고 반환값을 출력합니다.

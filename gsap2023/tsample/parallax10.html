소개

안녕하세요! 웹스토리보이입니다. 메뉴 클릭에 있어 메뉴 확대 축소 효과를 완성해보겠습니다. 일반적으로 메뉴가 있다가 스크롤을 내리면 축소되거나 모바일 메뉴로 바꾸는 형태로 많이 사용합니다. 다시 처음으로 돌아오면 크게 확대되는 효과입니다. 스크립트와 CSS를 같이 사용하는 효과입니다. 비교적 사용빈도가 높고 쉽게 때문에 쉽게 따라할 수 있습니다. 그럼 바로 시작해보겠습니다. 🤪

1. 기본 구조 만들기

1-1. 준비하기

GSAP를 배우는 시간이기 때문에 HTML/CSS 코딩은 생략하겠습니다. 기본 코딩은 복사해서 사용하겠습니다. 우선 웹폰트를 설정하고 GSAP에서 필요한 파일을 미리 셋팅해 놓겠습니다. GSAP는 자주 업데이트가 되기 때문에 제일 최선 버전을 사용하는게 좋습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>GSAP Scroll Effect</title>

    <!-- 웹폰트 설정 -->
    <link href="https://webfontworld.github.io/NexonLv1Gothic/NexonLv1Gothic.css" rel="stylesheet">
    <link href="https://fonts.googleapis.com/css2?family=Lato:wght@100&display=swap" rel="stylesheet">
</head>
<body>

    <!-- GSAP 라이브러리 설정 -->
    <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.11.5/gsap.min.js"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.11.5/ScrollTrigger.min.js"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.1/ScrollToPlugin.min.js"></script>
</body>
</html>

2. 스크립트 작업하기

2-1. 작업하기

우선 #parallax__nav 선택자에 클래스 is-active가 있을 때, 없을 때 CSS 작업을 먼저 해주어야 합니다. 물론 이것도 GSAP로 가능하지만 CSS로 간단하게 할 수 있기 때문에 CSS로 작업하겠습니다. 중요한건 소스가 깔끔하고 보기 좋은게 좋거든요! 그리고 ScrollTrigger.create에서 스타트 값과 끝나는 값을 설정합니다. 끝나는 값은 99999로 임의의 값을 설정했습니다. toggleClass 속성을 이용하면 간단하게 메뉴 확대 축소 효과는 끝이 납니다. 수고하셨습니다^^ 😄 

#parallax__nav li a {
    width: 80px; 
    transition: all 0.3s;
}
#parallax__nav.is-active li a {
    width: 30px; 
}

ScrollTrigger.create({
    start: "top -80",
    end: 99999,
    toggleClass: {
        className: "is-active", 
        targets: "#parallax__nav"
    }
});

3. 마무리

간단하게 CSS와 GSAP를 이용하여 작업해봤습니다. GSAP의 장점이 간단한 소스로 괜찮은 효과를 만들 수 있는게 매우 좋은 거 같습니다. 여러분들도 잘 따라해보고, 실제 사이트에 적용해 보면 아주 좋을 것 같습니다. 그럼 수고하셨습니다. 😱
